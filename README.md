# Christmas Bash Script
A small Bash script that prints an animated Christmas tree in the terminal ðŸŽ„ (with ANSI colors) and optionally plays a Christmas song using `mpg123`.

## Features
- Colored ASCII Christmas tree
- Multilingual greetings:
  - FELIZ NAVIDAD
  - MERRY CHRISTMAS
  - FROHE WEIHNACHTEN
- Optional background music playback via `mpg123`
- Prompts you for a tree height (with validation)

---

## Requirements

- **Bash** (Linux / macOS with Bash installed)
- (Optional) **mpg123** to play music
- (Optional) An MP3 file named `christmas-song.mp3` in the same folder as the script

### Install mpg123

**Ubuntu / Xubuntu / Debian:**
```bash
sudo apt update
sudo apt install mpg123
```
